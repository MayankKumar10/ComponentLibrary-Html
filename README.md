## Deployment

To Install Vision UI into your project

- You can install by copy the CDN link into
  your index.html file or.
- You can install by copy the CDN link into
  your root directory.

```bash
  <link rel="stylesheet" href="https://vision-ui-lib.netlify.app/styles/root.css">
```
