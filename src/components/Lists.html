List

lists allow web developers to group a set of related items in lists.

Unordered and Ordered Lists

An unordered and Ordered list starts with the <ul> and <ol> respectively tag. Each list item starts with the <li> tag. The list items will be marked with bullets and number by default respectively

  • Coffee
  • Tea
  • Milk
  1. Coffee
  2. Tea
  3. Milk

<ul>
<li>Coffee</li>
<li>Tea</li>
<li>Milk</li>
</ul>

<ol>
<li>Coffee</li>
<li>Tea</li>
<li>Milk</li>
</ol>

Stacked Lists

An Stacked list are nested lists,it starts with the <ul> tag. Each list item starts with the <li> and <li> tag. The list items will be marked with bullets by default

  • Coffee
  • Tea
    • Black tea
    • Green tea
  • Milk

<ul>
<li>Coffee</li>
<li>Tea
<ul>
<li>Black tea</li>
<li>Green tea</li>
</ul>
</li>
<li>Milk</li>
</ul>
